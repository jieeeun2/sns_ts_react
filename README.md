## 커밋 메세지

### 커밋 메세지 태그 규칙

| 태그명 | 용도                   |
| ------ | ---------------------- |
| chore  | 프로젝트 전반적인 설정 |
| build  | 패키지 설치            |
| feat   | 기능 추가, 변수명 변경 |
| fix    | 잘못된 부분의 수정     |
| design | css관련 UI디자인       |
| style  | 코드 포매팅            |
| remove | 파일 삭제              |
| rename | 파일명, 폴더명 변경    |
| docs   | 문서 수정              |

### 한 커밋당 하나의 작업단위, 한줄로 간결하게

추가적으로 여러줄 쓰고싶을때는 우선 한줄로 간결하게 쓴다음 줄의 끝에 `(+)`로 여러줄 글임을 표시
